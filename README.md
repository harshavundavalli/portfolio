# Harsha Vardhan Chowdary Vundavalli — Portfolio

A personal portfolio website built with React, Vite, and Tailwind CSS. Features a responsive design, animated sections, and a working contact form powered by EmailJS.

## Live Demo

[harsha-portfolio.vercel.app](https://harshavundavalli.vercel.app)

## Tech Stack

- **React** + **Vite**
- **Tailwind CSS**
- **EmailJS** — contact form email delivery

## Sections

- Hero
- About
- Skills
- Experience
- Projects
- Education
- Contact

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
